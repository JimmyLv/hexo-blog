JooMe：WiFi 分享中的无限可能

发表于 2015-10-10

JooMe：WiFi 分享中的无限可能

[核心提示] 始于 WiFi 开放共享运动，JooMe 已经成为有效的商户数据挖掘工具，并希望成为一个开放平台，一个日常生活中的基础设施。

在前两周的“每周一品”栏目中我们分别关注了 The Magazine 和 Circa ，这两个产品着眼于移动互联网时代的阅读体验。这一周，让我们谈谈 JooMe ，它着眼于移动互联网时代的基础设施——WiFi 网络。

JooMe 团队在其网站上这样描述自己的产品——

JooMe 是一个全球化的 WiFi 开放运动，旨在鼓励互联网使用者分享闲置带宽给周围的人免费使用，并由此获取一定的回报。与以往的 WiFi 分享项目不同的是，JooMe 对使用者没有任何收费或回报带宽分享的限制。所有人都可以自由使用 JooMe 提供的免费 WiFi。

可见在 JooMe 团队看来，JooMe 首先是一场 WiFi 开放运动，通过安装了 JooMe 定制固件的路由器，让加入 JooMe 运动的人们分享自己的 WiFi，连入他人分享的 WiFi，甚至获得更多其它回报。

在“开放、分享与回报”的理念下，JooMe 团队推动着这项运动的发展，并逐步找到了合适的方式，挖掘出其中的商业价值。

开放的 WiFi 与商家的需求

作为一个商家，最重要的利益诉求有两个：带来新客人，留住老客人。

——朱峰，JooMe 联合创始人

商户是 JooMe 创业初期的重点客户群，大多数情况下，商户们付出成本为消费者提供 WiFi，却并没有从 WiFi 服务中获得足够的利益。在 JooMe 联合创始人朱峰看来，商家最重要的利益诉求是两个：获得新用户，留住老用户。而 JooMe 就可以通过接入自己的免费 WiFi 服务很好地满足这两个需求：

  • 由于联网设备与用户之间存在对应关系，因此当新用户进店时，只要连上 JooMe 的免费 WiFi 热点，JooMe 就会根据联网设备的属性建立用户档案。这样的做法，比发放会员卡更加简单直接，几乎没有任何人工步骤的参与。
  • 当老用户再次进店连上 JooMe 的 WiFi 热点时，店方可以立刻收到提醒，看到这名老用户过往的消费时间、行为习惯等各种相关数据。进行针对性的服务。

除了这两个基本需求之外，JooMe 的 Wi-Fi 分享方案还可以为商家带来更多其它的便利——

  • 在用户连上 JooMe 的免费 Wi-Fi 开始上网之前会有一个页面出现，供用户注册、登陆或者直接开始免费上网。商家可以自定义这个页面上的多个部分，强化自己的品牌形象、推介当日的优惠信息等等。
  • JooMe 为商家提供管理后台，可以让商家看到基于 WiFi 连接行为的客流统计、客流报表和其它营销相关的参考数据，甚至已经开发出了相当精准的算法可以预测未来数日的客流量。在采访中朱峰告诉我，目前已经有一些接入 JooMe 免费 Wi-Fi 的大型连锁商家用 JooMe 方面提供的统计信息考核各店店长的工作表现。
  • JooMe 的提供的 WiFi 热点没有密码，登陆和注册过程也并非强制。让“反复询问服务员 WiFi 密码”这种现象成为了历史，有效减轻商家工作人员的负担。
  • 在网络质量上，JooMe 用多种技术确保用户能够获得合理的网速，并优先保证商家办公网络的畅通运行。同时，JooMe 还可以根据具体情况和商家需求提供实名接入认证等服务。免除了商家在网络搭建、维护和责任承担方面的后顾之忧。

正是因为对商家需求的这种精准把握和满足，JooMe 的服务推广进展很快。虽然目前仍然处在测试期，但是你已经可以在北京的多个热门商圈和大型连锁商家连上 JooMe 的免费 WiFi 信号。

在这样的情况下，商业模式也顺理成章地初步形成。在数据的积累和数据中蕴藏信息的挖掘上，JooMe 将为商户们提供多种增值服务：目前商家们已经可以在各自的管理后台看到不少统计信息，而更多、更加详尽、更高定制性的数据信息和分析报告，则需要商家付费获取。

用户的分享与切实的回报

尽管当前阶段商家群体是 JooMe 的主要业务重点，但是 JooMe 也一直保持着对普通个人用户的关注。在 JooMe 的首页上，我们可以看到使用 JooMe 路由器的用户已经在多个省份出现。而个人用户能够从 JooMe 开放 WiFi 运动中获取到的，也会是切实的回报。

虽然连上 JooMe 网络的用户不注册账号也可以直接联网，但是拥有账号之后 JooMe 用户可以获得更多的便利。例如在用户本人同意的情况下，其消费信息将会被记入商户的管理后台，当用户再次光顾同一商户时，接待者可以看到这一账号之前的消费记录，用户可以获得更精准的服务。

在 JooMe 的计划中，未来向个人用户大规模开放 JooMe 路由器使用和闲置带宽分享之后，个人用户将可以从在网时长、分享时间等诸多因素中累计自己的积分和等级，不同等级的用户可以获得各种不同的优惠。而这些优惠服务，在用户踏入相应商户时甚至不需要取出手机出示凭证：进入商户之后手机会自动连上店内的 JooMe WiFi，商家会自动得到用户进店信息和需要提供的优惠服务信息。

这样的场景是否似曾相识？是的，移动支付巨头 Square 用 iOS 的 GPS 地理围栏特性来完成同样的自动识别功能 。在 GPS 普遍偏移严重的我国，这种基于 WiFi 连接的位置判断和服务，比大多数基于 GPS 判断的 LBS 服务要更加精确和靠谱些。

JooMe 的未来——彻底的开放，与平台化的梦想

从创业开始，JooMe 团队就怀有开放的心态和平台化的梦想。

在采访中朱峰提到，JooMe 团队从 2012 年初开始创业，到目前为止看起来进度并不特别快。这主要是因为团队在服务的开放性上下了很大的功夫，在最初就考虑到了各种 API 接口的开放。

在路由器端，JooMe 为硬件厂商开放 API，他们可以在自己的产品中使用 JooMe 的固件，或者在自己的固件中置入 JooMe 的功能。在满足用户需求的同时，也可以根据设备销量和固件的装机量从 JooMe 团队获得一定的分成。而固件爱好者和普通路由器用户则可以在自己的路由器上刷入 JooMe 的固件，参与到 JooMe 的 WiFi 开放运动中来。

JooMe 的 API 还会开放给各种网站和手机应用，在获得用户同意的情况下，网站和应用可以调取用户的精确位置信息和行为习惯信息，为用户提供更好的服务。

甚至，JooMe 产品体系的服务器端和连接 WiFi 之后的入口页面接口也是开放的，JooMe 团队可以根据客户需求进行相应的定制，提供一整套 WiFi 接入和控制服务的解决方案，这样的定制服务已经有了成功的案例。

在 JooMe 团队的规划中，未来的 JooMe 将不止是一个开放 WiFi 运动和基于其上的商户信息管理平台。在更大的愿景中，JooMe 将会提供的是一个起着“基础设施”作用的开放平台：大量的用户使用 JooMe 路由器分享自己的带宽，大量的线下商户和团购网站等线上服务接入 JooMe 的开放 WiFi 体系中。在 JooMe 的平台上，用户走到各处都可以找到和享用免费的 JooMe WiFi，线上线下的商户则可以发起各种优惠、团购、打折等活动促销自己的服务和商品，同时获得精确的用户数据优化自己的服务。

想象有一天，你早上醒来拿起手机连上自己家的 JooMe 路由器，可以下载到附近咖啡店的折扣券；走在路上可以随时连入他人的 JooMe 路由器看看资讯发发微博；而当你踏入那家咖啡店走到柜台前时，因为连入了咖啡店的 JooMe 免费 WiFi，服务员已经知道你是他们的常客，你的折扣券内容，和你过去对咖啡的偏好。一句话也不用说，你就可以获得优惠的价格和满意的服务。

这便是 WiFi 分享中蕴藏的无限可能。我们期待 JooMe 能够将它一步步变成现实。
